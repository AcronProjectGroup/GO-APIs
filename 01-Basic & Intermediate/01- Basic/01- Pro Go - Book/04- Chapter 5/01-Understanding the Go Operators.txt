

+, -, *, /, %          --> These operators are used to perform arithmetic using numeric values

==, !=, <, <=, >, >=   --> These operators compare two values

||, &&, !              --> These are the logical operators

=, :=                  --> These are the assignment operators.

-=, +=, ++, --         --> These operators increment and decrement numeric values

&, |, ^, &^, <<, >>    -->  These are the bitwise operators, which can be applied to integer values.
