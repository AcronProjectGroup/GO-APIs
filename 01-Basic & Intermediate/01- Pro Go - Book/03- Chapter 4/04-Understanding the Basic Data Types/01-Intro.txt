



int = a whole number -- positive or negative.


uint = a positive whole number.
       size is platformdependent and will be either 32 or 64 bits.


byte = an alias for uint8 - used to represent a byte of data.


float32, float64 = numbers with a fraction.


complex64, complex128 = numbers that have real and imaginary components. 


bool = True/False


string = a sequence of characters


rune = a single Unicode code point. alias for int32.
